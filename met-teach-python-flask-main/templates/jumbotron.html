What is Flask?

Flask is a web framework. It makes creating websites a lot easier for developers. Flask simplifies working with urls (routing) and sending data between web pages.

Flask Documentation

What is Bootstrap?

Bootstrap is a CSS framework. Developers use Bootstrap to create professional looking websites which are visually pleasing.

Bootstrap Documentation

Flask Templates

Rather than generate HTML with Python the pages of this website use templates. Templates are a good way to mix simple HTML code with dynamic Python code.

Flask Routing

When you go to a web page your computer sends a GET or POST request to the web server where the web page lives.

HTTP Request Methods

Flask GET

The link below sends the data "World" to the /hello/<name> route of this website's Python file, app.py. As the route does not have a request method specified it assumes the GET method.

GET

Flask POST

It is common to use the POST method when sending data using HTML Forms. The Form below sends an email address to the /email route in app.py. Setting type="email" and required in the Form's HTML Input tag enables Bootstrap validation

Flask JSON (read local file)

JSON is the default format used to send data on the Web. The /json/local route reads JSON data from a text file called users.json in the sites's data folder.

Read local JSON File

Flask JSON (add to existing list)

The /json/users route in app.py shows a list of users in the website's memory. This list includes names from the users.json file and any names added using the HTML form. Note how the variable users is an empty, global list. It also worth noting the/json/users/add route redirects back to /json/users.

Add users to Website's Memory

Flask JSON (read remote file)

You often retrieve JSON data from a remote website, usually an API. The /json/remote route reads JSON data from the Github API.

Read remote JSON file

Flask - Random Duck

The /duck route in app.py gets JSON data from the API found at https://random-d.uk/api. The JSON data is then used to display a random image of a duck.

GET Random Duck

Flask - Duck Pagination

The /duck route in app.py gets JSON data from the API found at https://random-d.uk/api. The JSON data is then used to paginate through all the pictures of ducks available from the API.

GET Ducks and Paginate
